# Excel file Upload and Import using MERN(MySQL) 
# How to use
## start the backend app
- cd into "be" folder
- run "npm install" command
- run "npm run dev" command
## start the frontend app 
- cd into "fe" folder
- run "npm install" command
- run "npm start" command
